## Tabla Course
Se usa para almacenar los cursos
```go
type Course struct {
	ID                      uint      `json:"id" gorm:"primary_key"`
	Name                    string    `json:"name"`
	Description             string    `json:"description"`
	StartDate               time.Time `json:"start_date"`
	EndDate                 time.Time `json:"end_date"`
	Price                   float32   `json:"price"`
	ResolutionAuthorization string    `json:"resolution_authorization"`
	DateExam                time.Time `json:"date_exam"`

	CourseStudents []CourseStudent `json:"course_students"`
}
```
## Campos
- **Name(requerido)** Campo par ingresar el nombre del curso
- **Description(opcional)** Descripcion corta del curso que se llevara
- **StartDate(requerido)** Fecha de inicio del curso
- **EndDate(requerido)** Fecha de finalizacion del curso
- **Price(opcional)** Precio que se establece al curso
- **ResolutionAuthorization(opcional)** Campo par ingresar la resolucion de atorizacion del curso
- **DateExam(opcional)** Fecha estimado para el examen final del curso
