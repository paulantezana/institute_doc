## Tabla 
Se usa para almacenar las configuraciones generales del instiutto
*en esta tabla solo existe un registro*
```go
type Setting struct {
	ID                         uint   `json:"id" gorm:"primary_key"`
	Logo                       string `json:"logo"`
	Ministry                   string `json:"ministry"`
	NationalEmblem             string `json:"national_emblem"`
	Prefix                     string `json:"prefix"`
	PrefixShortName            string `json:"prefix_short_name"`
	Institute                  string `json:"institute"`
	MinHoursPracticePercentage uint   `json:"min_hours_practice_percentage"`
	YearName                   string `json:"year_name"`

	Director                   string `json:"director"`
	AcademicLevelDirector      string `json:"academic_level_director"`
	ShortAcademicLevelDirector string `json:"short_academic_level_director"`
	ResolutionAuthorization    string `json:"resolution_authorization"`
	ResolutionRenovation       string `json:"resolution_renovation"`
	ModularCode                string `json:"modular_code"`
	ManagementType             string `json:"management_type"`
	DreGre                     string `json:"dre_gre"`

	WebSite string `json:"web_site"`
	Email   string `json:"email"`
}
```

## Campos
- **Logo(opcional)** Campo para almacenar la url del logo de la institucion
- **Ministry(opcional)**  Campo para almacenar la url del encabezado del ministerio de educacion
- **NationalEmblem(opcional)** Campo para almacenar la url del escudo nacional del Peú
- **Prefix(opcional)**  Nombre largo del tipo de institucion
- **PrefixShortName(opcional)**  Nombre corto del tipo de institucion
- **Institute(opcional)** Campo para almacenar el nombre de la institucion
- **MinHoursPracticePercentage(opcional)**  Campo para almacenar el porcentaje de horas aceptables para realizar las revisiones de las practicas modulares
- **YearName(opcional)** Campo para almacenar el nombre del año actual
- **Director(opcional)** Campo para almacenar el nombre completo del director o directora
- **AcademicLevelDirector(opcional)** Campo para almacenar el nivel academinco del director o directora
- **ShortAcademicLevelDirector(opcional)** Campo para almacenar la abrebiacion del nivel academco del director o directora
- **ResolutionAuthorization(opcional)** Campo para almacenar la resolución de licenciamiento y/o autorización del isntiuto
- **ResolutionRenovation(opcional)** Campo para almacenar la resolución de renovación y/o revalidación del instituto
- **ModularCode(opcional)**  Campo para almacenar el codigo modular del isntituto
- **ManagementType(opcional)** Campo para almacenar el tipo de gestion del instituto si es una publica o privada
- **DreGre(opcional)** Campo para almacenar la dreg a la que pertenece el instituto
- **WebSite(opcional)** Campo para almacenar el sitio web del instituto
- **Email(opcional)** Campo para almacenar email del instituto
